# Changelog

All notable changes to Mana Hyderabad will be documented in this file.

This project follows a lightweight changelog format and is moving toward conventional commits for automated release notes.

## [0.1.0] - 2026-05-31

### Added

- Next.js citizen reporting flow with multilingual UI scaffolding.
- FastAPI complaint APIs with PostgreSQL and PostGIS persistence.
- Admin complaint list, detail, analytics, map, nearby complaints, and hotspot endpoints.
- Cloudinary-backed image upload endpoint and frontend photo upload flow.
- Interactive MapLibre location picker and admin operations map.
- Repository compliance docs, security policy, CI, formatting, tests, and Docker metadata.
