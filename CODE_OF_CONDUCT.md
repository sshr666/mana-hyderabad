# Code of Conduct

Mana Hyderabad contributors commit to a respectful, inclusive, and civic-minded collaboration environment.

## Expected Behaviour

- Use welcoming and professional language.
- Respect different backgrounds, identities, skill levels, and viewpoints.
- Give and receive feedback constructively.
- Protect citizen privacy and avoid sharing sensitive personal information.
- Clearly mark experimental AI or geospatial behaviour.

## Unacceptable Behaviour

- Harassment, threats, or discriminatory language.
- Publishing private contact details, credentials, or citizen complaint data.
- Deliberately misleading users about AI-generated or unverified outputs.
- Disruptive conduct that prevents productive collaboration.

## Reporting

Report concerns to the repository maintainers through a private GitLab issue or project contact channel. Maintainers should review reports promptly and take proportionate action, including warnings, issue moderation, or removal from the project.

## Scope

This code applies in project repositories, issues, merge requests, chat spaces, demos, and community events connected to Mana Hyderabad.
